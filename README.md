# radar-landing

Одностраничная презентация проекта Радар — для показа пилотам, друзьям, партнёрам.

Деплоится автоматически на Vercel при push в `main`.

## Структура

- `index.html` — главная и единственная страница
- `vercel.json` — конфигурация Vercel (статика без сборки)

## Локально

Открыть `index.html` в браузере двойным кликом, или:

```bash
python -m http.server 8000
# открыть http://localhost:8000
```

## Обновление

Просто отредактировать `index.html` и push в main. Vercel перезальёт страницу через 30 сек.
